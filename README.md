# UserManagement

Run The Project

You will need the following tools:

Vs Code
.Net Core 6 or later
Docker Desktop

Installing
1-Clone the repository
2-Docker Desktop should be running.
3-At the root directory which include docker-compose.yml files, run below command:
docker-compose build
docker-compose up -d
