# UserManagement

Run The Project

You will need the following tools:

 * VS Code
 * .Net Core 6 or later
 * Docker Desktop
 
<p align="left">
 - Clone the repository
</p>
<p align="left">
 - At the root directory which include docker-compose.yml files, run below command:
</p>
```csharp
docker-compose build
docker-compose up -d
```
